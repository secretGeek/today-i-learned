Guidelines

Each page of this site should be a factoid that you expect to refer to again.

Hence, any factoid that is merely "Well that's interesting!" but would never be referenced again is "trivia" and NOT wanted here. So they mostly end up being technical things (though the concepts and reading notes categories are an exception).

The folder structure means things are categorised by ONE main topic, not tagged with endless topics/subtopics.
