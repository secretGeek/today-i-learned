seqcli

Commandline tool for seq (see overview).

Supports logging (seqcli log), searching (seqcli search), tailing (seqcli tail), querying (seqcli query) and JSON or plain-text log file ingestion (seqcli ingest), and much more. â€” seqcli at github

Get it from datalust at https://github.com/datalust/seqcli

Or get it via dotnet

dotnet tool install --global seqcli

> seqcli -?
