update bower

To update bower one simply has to...

npm i -g bower

where:

  • i is for install and
  • g is for global

output like:

C:\Users\UserName\AppData\Roaming\npm\bower -> C:\Users\UserName\AppData\Roaming\npm\node_modules\bower\bin\bower
C:\Users\UserName\AppData\Roaming\npm
+-- bower@1.8.0
